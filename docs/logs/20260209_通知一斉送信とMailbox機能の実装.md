# 通知一斉送信とMailbox機能の実装

## 作業日時
2026-02-09

## 作業概要
StaffがMembership有効期限に基づいてUserへメッセージを一斉送信できる機能と、User側のMailbox（受信箱）機能、FCMプッシュ通知機能を実装した。

## 実施内容

### 1. バックエンド - 新規モデル追加

**ファイル**: `wellbee/attendances/models.py`

- **NotificationDaysSetting**: 通知日数設定（例: 30日前、14日前）。`days_before_expire`（ユニーク）と`message`を保持
- **UserMessage**: ユーザー受信メッセージ。`user`, `title`, `content`, `is_read`, `read_at`を保持。Mailboxの実体
- **UserFCMToken**: FCMデバイストークン管理。`user`, `token`, `device_type`を保持

### 2. バックエンド - API追加

**ファイル**: `wellbee/attendances/serializers.py`, `views.py`, `urls.py`

| エンドポイント | 用途 |
|---|---|
| `notification-settings/` | 日数設定のCRUD（Staff用） |
| `notification-settings/{id}/target-users/` | 対象ユーザープレビュー |
| `notification-settings/{id}/send/` | メッセージ一斉送信 + FCMプッシュ通知 |
| `mailbox/` | 自分のメッセージ一覧（User用） |
| `mailbox/{id}/read/` | メッセージ既読化 |
| `mailbox/unread-count/` | 未読件数取得 |
| `fcm-token/register/` | FCMトークン登録 |
| `fcm-token/unregister/` | FCMトークン無効化 |

### 3. バックエンド - FCM送信ユーティリティ

**ファイル**: `wellbee/attendances/utils.py`（新規作成）

- `send_push_notification()`: Firebase Admin SDK経由でマルチキャスト送信
- 500件ずつバッチ処理、無効トークンの自動無効化

### 4. バックエンド - マイグレーション

**ファイル**: `wellbee/attendances/migrations/0017_notificationdayssetting_usermessage_userfcmtoken.py`（自動生成）

### 5. Flutter - Firebase/FCM初期化

**変更ファイル**:
- `pubspec.yaml`: `firebase_core`, `firebase_messaging` 追加
- `lib/firebase_options.dart`: コメントアウト解除・有効化
- `lib/main.dart`: Firebase初期化とバックグラウンドメッセージハンドラ追加

**新規ファイル**:
- `lib/services/fcm_service.dart`: FCMトークン取得・登録・更新・解除

### 6. Flutter - User側 Mailbox画面

**新規ファイル**:
- `lib/screens/mailbox/mailbox_list.dart`: メッセージ一覧画面（既読/未読の色分け表示）
- `lib/screens/mailbox/mailbox_detail.dart`: メッセージ詳細画面（表示時に自動既読化）

**変更ファイル**:
- `lib/screens/home.dart`: 右上にMailboxアイコン追加。未読件数を赤バッジで表示。FCM初期化とフォアグラウンド通知リスナーも追加

### 7. Flutter - Staff側 通知管理画面

**新規ファイル**:
- `lib/screens/staff/notification/notification_home.dart`: 通知機能ホーム（日数設定管理 / メッセージ送信への導線）
- `lib/screens/staff/notification/notification_settings_list.dart`: 日数設定一覧。FABで新規作成、タップで編集
- `lib/screens/staff/notification/notification_setting_edit.dart`: 日数設定の作成・編集・削除。日数、メッセージ、有効/無効を設定
- `lib/screens/staff/notification/notification_send.dart`: 送信画面。日数設定選択 → 対象者プレビュー → 送信実行の3ステップ

**変更ファイル**:
- `lib/screens/staff/staff_home_page.dart`: importを`notification_home.dart`に変更、`NotificationHomePage`へ遷移
