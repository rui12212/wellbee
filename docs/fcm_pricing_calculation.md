# Firebase Cloud Messaging (FCM) 料金計算書

## 結論

**FCMは完全無料です。** 月間50ユーザー × 5回通知 = 250通/月 のプッシュ通知は、**0円**で運用できます。

---

## FCM料金体系

### 公式料金

| 項目 | 料金 |
|------|------|
| メッセージ送信 | **無料（無制限）** |
| iOS/Android/Web対応 | **無料** |
| ターゲティング・セグメント | **無料** |
| 分析・レポート | **無料** |

### 制限事項

| 制限 | 値 | 影響 |
|------|-----|------|
| 1分あたりの送信上限 | 600,000メッセージ | 50人規模では影響なし |
| 1デバイスあたりの上限 | なし | - |
| 月間上限 | なし | - |

**出典:** [Firebase Pricing](https://firebase.google.com/pricing), [FCM Throttling and Quotas](https://firebase.google.com/docs/cloud-messaging/throttling-and-quotas)

## 想定ケースの計算

### 条件
- 月間アクティブユーザー: **50人**
- 通知タイミング: **1日前、3日前、7日前、14日前、30日前**（5回）
- 1ユーザーあたり1つのMembershipと仮定

### 月間通知数

| 項目 | 計算式 | 数値 |
|------|--------|------|
| 1ユーザーあたりの通知回数 | 5回/会員期限 | 5通 |
| 月間ユーザー数 | - | 50人 |
| **月間合計通知数** | 50 × 5 | **250通** |
| **年間合計通知数** | 250 × 12 | **3,000通** |

### 月間コスト

| 項目 | 料金 |
|------|------|
| FCMメッセージ送信 | ¥0 |
| **合計** | **¥0** |

## 関連するFirebaseサービスの料金（参考）

FCM自体は無料ですが、連携する他のFirebaseサービスを使用する場合は料金が発生する可能性があります。

### 今回の実装で使用するサービス

| サービス | 用途 | 料金 |
|---------|------|------|
| Firebase Cloud Messaging | プッシュ通知送信 | **無料** |
| Firebase Admin SDK（サーバー） | 通知送信API | **無料** |

### 使用しないサービス（参考）

| サービス | 料金 | 備考 |
|---------|------|------|
| Cloud Firestore | 読み書き課金 | 今回は使用しない（Django使用） |
| Cloud Functions | 実行回数課金 | 今回は使用しない（Django使用） |
| Firebase Authentication | 無料（SMS認証は有料） | 今回は使用しない |

## 隠れコストの確認

### 発生しないコスト

| 項目 | 理由 |
|------|------|
| Firestore読み取り | Djangoバックエンドを使用するため不要 |
| Cloud Functions実行 | Djangoで直接FCM APIを呼び出すため不要 |
| SMS認証 | 電話番号+パスワード認証のため不要 |

### 発生する可能性のあるコスト

| 項目 | 条件 | 想定料金 |
|------|------|---------|
| サーバー費用 | 既存Djangoサーバーで対応 | 追加なし |
| Firebase Admin SDK | pip install のみ | ¥0 |

## 結論

### 総コスト: **月額 ¥0**

Firebase Cloud Messagingは完全無料のサービスであり、今回の実装（月間50ユーザー × 5回通知）においてFirebase関連の追加費用は**一切発生しません**。

### 推奨事項

1. **FCMで問題なし** - コスト面での懸念は不要
2. **スケーラビリティ** - 将来ユーザー数が増加しても無料
3. **代替案不要** - 有料のSMS通知などを検討する必要なし
