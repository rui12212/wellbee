# AWS費用・DBクエリ数 試算

## 前提条件

| 条件 | 値 |
|------|-----|
| 月間アクティブユーザー | 50人 |
| 通知タイミング | 30日前・14日前・7日前・3日前・1日前・当日（計6回） |
| アーキテクチャ | Django on ECS (Fargate) / MySQL on EC2 |
| リージョン | アジアパシフィック（東京）ap-northeast-1 |
| ユーザーのアプリ起動回数 | 1日2回（平均） |
| FCM通知開封率 | 70% |
| 為替レート | 1 USD = 150 JPY |

> **注意**: 料金はAWSの公式サイトで最新情報を確認してください。本試算はナレッジカットオフ時点（2025年8月）の単価をもとにした参考値です。

---

## Part 1: 月間DBクエリ数

### 1-1. Staff側：通知送信クエリ

Staff が6つの日数設定をそれぞれ毎日トリガーする想定（0件マッチでもクエリは実行される）。

**`send` アクション 1回あたりのクエリ内訳:**

| # | 処理 | クエリ数 |
|---|------|---------|
| 1 | `NotificationDaysSetting.objects.get(pk=id)` | 1 |
| 2 | `Membership.objects.filter(expire_day=target_date, ...).select_related(...)` | 1（JOINで1本） |
| 3 | `UserMessage.objects.bulk_create([最大50件])` | 1（INSERT一括） |
| 4 | `UserFCMToken.objects.filter(user_id__in=[...], is_active=True)` | 1 |
| **小計** | | **4クエリ/回** |

```
6設定 × 4クエリ × 30日 = 720 クエリ/月
```

---

### 1-2. User側：Mailbox関連クエリ

| 処理 | 計算式 | クエリ数/月 |
|------|--------|------------|
| ホーム画面 未読件数取得（`unread-count`） | 50人 × 2回/日 × 30日 × 1クエリ | 3,000 |
| Mailbox一覧表示（FCM通知タップ） | 50人 × 6通知 × 70%開封 = 210回 × 1クエリ | 210 |
| メッセージ既読化（`mark_as_read`） | 210回 × 2クエリ（get_object + UPDATE） | 420 |

```
User側合計 = 3,000 + 210 + 420 = 3,630 クエリ/月
```

---

### 1-3. 月間クエリ合計

| カテゴリ | クエリ数/月 |
|---------|------------|
| Staff側 通知送信 | 720 |
| User側 未読件数取得 | 3,000 |
| User側 Mailbox一覧 | 210 |
| User側 既読化 | 420 |
| **合計** | **4,350 クエリ/月** |

> ※ 既存機能（予約・メンバーシップ取得など）のクエリは含まない

## Part 3: サマリー

| 指標 | 値 |
|------|-----|
| 月間DBクエリ数（通知機能） | **約4,350回** |
| 月額AWS費用（Scenario A / NAT Gateway有） | **約 $89 / ¥13,400** |
| 月額AWS費用（Scenario B / NAT Gateway無） | **約 $44 / ¥6,600** |

### コスト最適化のポイント

1. **NAT Gateway を避ける（最大削減）**
   ECSタスクをパブリックサブネットに置き、セキュリティグループでMySQLへの直接アクセスを制限すれば 約 **-¥6,800/月**

2. **Fargate Spot の活用**
   Djangoサーバーは通常タスクではなく Fargate Spot（最大70%割引）を使うと **-¥1,180/月** 程度

3. **ALB を省略してNLBまたはEC2上のNginxに変更**
   ALBの固定費 $17.74 を削減可能（ただしHTTPS証明書管理が複雑になる）

4. **EC2をRDSに移行した場合の追加コスト**
   `db.t3.micro` RDS: $0.026/時間 = **$18.98/月** （EC2 MySQLより約 +$7/月 だがバックアップ・パッチが自動化される）
